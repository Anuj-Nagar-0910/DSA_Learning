🔍Intuition

Tree = Family Tree

One ancestor (root) → children → grandchildren. No cycles. Every node has exactly one parent (except root).

HierarchicalOne parentAcyclic

Used in: DOM, file systems, org charts, decision trees, BSTs, heaps

Graph = City Road Map

Cities = nodes, roads = edges. Roads can go one-way or both ways. Can have cycles. No strict parent-child rule.

GeneralAny connectionsMay have cycles

Used in: social networks, GPS routing, web crawlers, dependency graphs

Key Mental Model

Tree is a restricted Graph

  • Connected + Acyclic + Directed (parent→child)
  • N nodes → exactly N−1 edges
  • Always one path between any two nodes

Graph is the general case

  • Can be connected or disconnected
  • Can have cycles
  • Edges can be weighted

🐍Implementation

Complete BST Implementation

class BST:
    def __init__(self):
        self.root = None

    def insert(self, val):
        self.root = self._insert(self.root, val)

    def _insert(self, node, val):
        if not node:
            return TreeNode(val)
        if val < node.val:
            node.left = self._insert(node.left, val)
        elif val > node.val:
            node.right = self._insert(node.right, val)
        return node

    def search(self, val):
        return self._search(self.root, val)

    def _search(self, node, val):
        if not node or node.val == val:
            return node
        if val < node.val:
            return self._search(node.left, val)
        return self._search(node.right, val)

Graph — Build & Traverse

from collections import defaultdict, deque

class Graph:
    def __init__(self):
        self.adj = defaultdict(list)

    def add_edge(self, u, v, directed=False):
        self.adj[u].append(v)
        if not directed:
            self.adj[v].append(u)

    def dfs(self, start):
        visited, stack = set(), [start]
        result = []
        while stack:
            node = stack.pop()
            if node not in visited:
                visited.add(node)
                result.append(node)
                stack.extend(self.adj[node])
        return result

    def bfs(self, start):
        visited, queue = {start}, deque([start])
        result = []
        while queue:
            node = queue.popleft()
            result.append(node)
            for n in self.adj[node]:
                if n not in visited:
                    visited.add(n)
                    queue.append(n)
        return result

# Usage
g = Graph()
g.add_edge('A', 'B')
g.add_edge('A', 'C')
g.add_edge('B', 'D')
print(g.dfs('A'))   # ['A', 'C', 'B', 'D']
print(g.bfs('A'))   # ['A', 'B', 'C', 'D']

Detect Cycle in Undirected Graph

def has_cycle(graph, n):
    visited = [False] * n
    def dfs(node, parent):
        visited[node] = True
        for neighbor in graph[node]:
            if not visited[neighbor]:
                if dfs(neighbor, node):
                    return True
            elif neighbor != parent:
                return True  # back edge = cycle!
        return False
    return any(dfs(i, -1) for i in range(n) if not visited[i])

🧩Patterns & Use Cases

Pattern: Two-Pass DFS

Run DFS twice to find connected components, bridges, or articulation points (Kosaraju's / Tarjan's)

Pattern: BFS + Distance Array

Maintain dist[] alongside BFS for shortest unweighted paths. Initialize with -1 or infinity.

Pattern: Topological Sort

DAG: use DFS post-order OR Kahn's algorithm (BFS + in-degree). Use for task scheduling, build order.

Pattern: Union-Find

Track connected components efficiently. Use for Kruskal's MST, cycle detection in undirected graphs.

Pattern: Level BFS on Trees

Use a sentinel None or size-based loop to process level by level. Useful for zigzag, average, or right-view problems.

Pattern: Lowest Common Ancestor

Use post-order DFS. Return node when both left and right subtrees find the targets.

Dijkstra's — Clean Template

import heapq

def dijkstra(graph, start):
    dist = {node: float('inf') for node in graph}
    dist[start] = 0
    heap = [(0, start)]  # (distance, node)

    while heap:
        d, u = heapq.heappop(heap)
        if d > dist[u]:
            continue
        for v, w in graph[u]:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                heapq.heappush(heap, (dist[v], v))
    return dist

3. Lowest Common Ancestor of BST

def lowestCommonAncestor(root, p, q):
    while root:
        if p.val < root.val and q.val < root.val:
            root = root.left
        elif p.val > root.val and q.val > root.val:
            root = root.right
        else:
            return root

4. Course Schedule (Cycle Detection in Directed Graph)

def canFinish(numCourses, prerequisites):
    graph = defaultdict(list)
    for a, b in prerequisites:
        graph[b].append(a)
    # 0=unvisited, 1=in-progress, 2=done
    state = [0] * numCourses
    def dfs(node):
        if state[node] == 1: return False  # cycle!
        if state[node] == 2: return True
        state[node] = 1
        for nei in graph[node]:
            if not dfs(nei): return False
        state[node] = 2
        return True
    return all(dfs(i) for i in range(numCourses))

🚀Interview Tips & Common Mistakes

Top Mistakes to Avoid

  • Forgetting the visited set in graph traversal → infinite loop in cyclic graphs
  • Confusing height vs depth — height measured from leaves up, depth from root down
  • Assuming BST is always balanced — O(log n) is average case, not guaranteed
  • Not handling disconnected graphs — loop over all nodes, not just from one source
  • Mutating the graph mid-traversal — use a copy if you mark cells in place
  • Stack overflow with deep recursion — convert to iterative DFS for production

Quick Interview Checklist

Before you code

  • Directed or undirected?
  • Weighted edges?
  • Can there be cycles?
  • Connected or disconnected?
  • What's the expected output — path, bool, count?

Optimization signals

  • Shortest path → BFS (unweighted), Dijkstra (weighted)
  • All paths → DFS + backtrack
  • Ordering → Topological sort
  • Components → Union-Find or DFS loop
  • Balance needed → AVL or heapq

Python-specific tips

  • Use collections.deque for BFS — O(1) popleft vs O(n) for list
  • Use collections.defaultdict(list) for adjacency lists
  • Use heapq (min-heap) for Dijkstra — negate values for max-heap
  • Increase recursion limit: sys.setrecursionlimit(10**5) for deep trees
  • Use set() for visited — O(1) lookup
