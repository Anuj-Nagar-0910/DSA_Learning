Part 2 — the tree height IS log n

Each level of the tree doubles the number of nodes. So the height grows as the inverse of doubling — which is log₂.

Part 3 — why insert is O(log n)

When you insert into a heap, the new element bubbles up from the bottom to its correct position. The maximum distance it can travel = the height of the tree = log₂(n).
