Basics Folder

FindMaxInArray.pyheap

Concepts: max heap behavior with Python's min-heap, insert, pop maximum.

Implements max-removal queries by storing negative values in heapq.

  • Good DSA idea: converts a min-heap into max-heap behavior.
  • Next step: wrap shared heap state in a class to match the prompt cleanly.
basic_linked_list.pylinked list

Concepts: nodes, head pointer, append, prepend, delete, traversal.

Builds a beginner-friendly singly linked list with detailed method explanations.

  • Strong learning value: pointer updates are explained line by line.
  • Append is O(n); adding a tail pointer would make it O(1).
QueueSimple.pyqueue

Concepts: FIFO, deque, O(1) append and popleft.

Uses collections.deque correctly for queue operations.

  • Good library choice for front removals.
  • Ready for platform stubs that expect insert and getFirst.
MostFrequent.pyreview

Concepts: frequency map, ASCII tie-breaking.

Counts characters with a dictionary and attempts to return the most frequent character.

  • Insight: the update line should assign max_count = count, not count = max_count.
  • A focused test for ties like aaaaAAAA would catch this quickly.
learning_class.pyOOP

Concepts: classes, constructors, instance methods, class variables.

Practices object-oriented Python with a simple shared user database dictionary.

  • Good for understanding instance state versus class-level state.
  • Next step: avoid printing passwords in real applications.
__init__.pynotes

Concepts: package marker plus linked-list notes.

Currently stores explanatory notes inside the package initializer.

  • Insight: move learning notes into a markdown or HTML note so imports stay lightweight.

Intermediate Folder

AdditionOfMatrix.pymatrix

Concepts: nested list comprehension, matrix addition.

Adds two same-sized matrices element by element.

average_height.pyinput

Concepts: stdin parsing, floating-point average, formatted output.

Reads student heights and prints the average to six decimal places.

diagonalsum.pyreview

Concepts: square matrix traversal, principal diagonal.

Contains multiple diagonal-sum approaches, including a clean helper function.

  • Insight: the top-level input logic reads values inconsistently; keep one solve path to avoid blocked input.
fizzbuzz basic.pyloops

Concepts: divisibility checks, branching, output per line.

Classic FizzBuzz solution from 1 to n.

IntersectionOfTwoArrayBasic.pytwo pointers

Concepts: sorting, duplicate-aware intersection, two-pointer scan.

Finds the sorted intersection while preserving duplicate counts.

  • Good algorithmic structure after sorting both inputs.
  • Next step: remove repeated print lines in the main block.
LinkedListSplit.pylinked list

Concepts: stable partition, odd/even splitting, tail pointers.

Splits a linked list into odd and even lists while preserving original order.

  • Strong use of separate head and tail pointers.
  • Correctly terminates both result lists with None.
MaxProduct2.pyheap

Concepts: selecting top two values, max product.

Uses a heap to retrieve the two largest numbers before computing the product.

  • Insight: for this constraint, a single pass tracking two maximums would use O(1) extra space.
Recursion.pyrecursion

Concepts: recursive sum, factorial, Fibonacci.

Shows base cases and recursive calls through three classic examples.

  • Insight: Fibonacci is exponential without memoization, which is useful to compare against dynamic programming later.
SearchTarget.pybinary search

Concepts: sorted arrays, left/right bounds, O(log n) search.

Implements binary search and returns the target index or -1.

simplediampond.pypatterns

Concepts: range loops, string multiplication, output shaping.

Builds a simple diamond pattern with increasing and decreasing rows.

ValidParanthesies.pystack

Concepts: stack, bracket matching, early invalid detection.

Validates bracket sequences using a list-backed stack and closing-to-opening map.

WordCount.pyset

Concepts: splitting strings, uniqueness with sets.

Counts unique whitespace-separated words in a string.
